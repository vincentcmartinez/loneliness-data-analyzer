A program that parses data from Dartmouth's StudentLife survey, available for download here: https://studentlife.cs.dartmouth.edu/dataset.html
The program takes GPS data to estimate subjects' average time spent at home during the 10-week period of the study, and compares it to the scores of their loneliness survey. 
this is an ad-hoc solution, but some code snippets may be helpful for use in related projects.
